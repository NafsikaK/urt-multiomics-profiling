# URT Multiomics Profiling

Code repository for analyses associated with the manuscript:

"Niche-specific immune and microbial signatures across the healthy upper respiratory tract mucosa"

## Overview

This repository contains R scripts used for data processing, statistical analyses, and figure generation for the manuscript.

The associated datasets are available at:
https://data.mendeley.com/preview/96nnvxsr32?a=07ef5f80-67fb-4c7d-95a7-dd7705580735


## Requirements

Analyses were performed in R (version 4.4.1).


## Reproducing analyses

Scripts can be run in any order
